Что готово на момент дедлайна:
1. Осуществляется переход с экрана Artist на экран Drawings и обратно с помощью Navigation Controller
1. Отрисованы кнопки с тенями на экране Artist и Drawings, могут отображаться в трех состояниях(default, highlited, disabled)
2. Timer View Controller реализован на языке Swift (открывается и закрывается по нажатию кнопок Open Timer и Save соответственно) + прокидывается значение времени на основной экран с помощью делегата TimerVCDelegate(использовал мост Swift to ObjC).
